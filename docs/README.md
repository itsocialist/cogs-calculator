## About

COGS Calculator is a comprehensive cost-of-goods-sold calculator for cannabis edibles manufacturing, featuring:
- Real-time cost calculations
- Batch scaling
- Ingredients manifest with export
- Professional PDF generation
- Dark glass UI aesthetic
